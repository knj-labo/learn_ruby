**Q9. 以下のコードがあります。**

```
x = 0

4.times do |i|
  x += i
end

p x
```

**実行結果として正しいものを選択してください。(1つ選択)**

- `4` →　`6`
  - `times`メソッドが何をするのか理解していない
  - `Integer#times`メソッドは0から指定された値より1つ小さい整数までの値を順番にブロックパラメータとして渡します。
  - 0 += 1, 0 += 2, 0 += 3,　→　6
